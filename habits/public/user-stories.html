Authentication System

Core

User Registration

As a new user, I want to create an account with email and password.

User Login/Logout

As a registered user, I want to log in and log out of the system.

User Management

Core

User Profile

As a user, I want to view and update my profile information.

Role-Based Access

As an admin, I want to manage user roles and permissions.

Habits Integration

Integration

User Synchronization

As an admin, I want to sync users between local auth and Habits service.

Idempotent Sync

As a system, I want sync operations to be idempotent to prevent duplicates.
